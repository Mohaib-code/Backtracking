# Backtracking Algorithms in Java

This repository contains implementations of important **Backtracking problems** in Java. It is designed to help students understand recursion, decision trees, and problem-solving techniques in Data Structures and Algorithms (DSA).

---

## 📌 Topics Covered

- Introduction to Backtracking
- Types of Backtracking
- Backtracking on Arrays
- Find Subsets
- Find Permutations
- N-Queens Problem
  - Print all solutions
  - Count total ways
  - Print one solution
  - Time Complexity
- Grid Ways Problem

---

## 🚀 What is Backtracking?

Backtracking is a recursive algorithmic technique used to solve problems by trying all possible solutions and undoing choices that do not lead to a valid solution.

---

## 💡 Key Concepts

- Recursion
- Decision Making
- State Space Tree
- Undo (Backtrack Step)

---

## 🧠 Example Problems

### 1. Subsets
Generate all subsets of a given array.

### 2. Permutations
Find all possible arrangements of elements.

### 3. N-Queens
Place N queens on an N×N board such that no two queens attack each other.

### 4. Grid Ways
Count the number of ways to reach the bottom-right of a grid.

---

## ⚙️ Time Complexity

Backtracking problems generally have exponential time complexity:

- Subsets → O(2^n)
- Permutations → O(n!)
- N-Queens → O(n!)
